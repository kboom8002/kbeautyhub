# Canonical Question Set
Version: v1
Vertical: K-Beauty Skincare

---

## 1. 문서 목적

이 문서는 raw question을 운영 가능한 대표 질문으로 압축한 Canonical Question 세트를 정의한다.

---

## 2. ID 규칙

- `CQ-ENTRY-###`
- `CQ-FIT-###`
- `CQ-TYPE-###`
- `CQ-ING-###`
- `CQ-COMPARE-###`
- `CQ-USE-###`
- `CQ-PROOF-###`
- `CQ-BOUNDARY-###`
- `CQ-BRAND-###`
- `CQ-ACTION-###`

---

## 3. Core Canonical Questions

### ENTRY
- CQ-ENTRY-001 내 피부 고민에는 어떤 스킨케어 접근이 맞나?
- CQ-ENTRY-003 민감하고 쉽게 자극받을 때 어떤 루틴부터 시작해야 하나?
- CQ-ENTRY-004 장벽이 약해진 것 같을 때 무엇을 줄이고 무엇을 써야 하나?
- CQ-ENTRY-005 트러블과 피지가 함께 고민일 때 어떤 방향으로 루틴을 잡아야 하나?
- CQ-ENTRY-006 스킨케어 초보자는 몇 단계 루틴으로 시작하는 게 맞나?

### FIT
- CQ-FIT-001 이 제품군은 내 피부 타입에 맞나?
- CQ-FIT-003 이 제품은 지성 또는 복합성 피부에 맞나?
- CQ-FIT-004 이 제품은 민감 피부에 맞나?
- CQ-FIT-005 이 고기능 성분은 초보자도 시작할 수 있나?
- CQ-FIT-008 여러 피부 고민이 동시에 있을 때도 이 루틴이 맞나?

### TYPE
- CQ-TYPE-001 이 피부 고민에는 어떤 제품군부터 봐야 하나?
- CQ-TYPE-002 에센스, 세럼, 앰플은 무엇이 어떻게 다른가?
- CQ-TYPE-003 로션, 크림, 젤 크림은 무엇이 어떻게 다른가?
- CQ-TYPE-004 최소 루틴을 구성할 때 필수 단계와 선택 단계는 무엇인가?

### ING
- CQ-ING-001 이 성분은 어떤 피부 고민에 왜 쓰는가?
- CQ-ING-006 두 성분을 함께 써도 되는가?

### COMPARE
- CQ-COMPARE-001 두 제품군은 무엇이 어떻게 다른가?
- CQ-COMPARE-003 두 제품 중 내 피부 고민에는 어느 쪽이 더 맞나?
- CQ-COMPARE-004 장벽 크림과 수분 크림은 무엇이 다른가?
- CQ-COMPARE-005 비타민C와 나이아신아마이드는 어떤 상황에서 다르게 선택해야 하나?

### USE
- CQ-USE-001 제품은 어떤 순서로 바르는가?
- CQ-USE-002 이 성분 또는 제품은 얼마나 자주 써야 하나?
- CQ-USE-004 자극을 줄이기 위한 도입/적응 스케줄은 어떻게 잡아야 하나?
- CQ-USE-005 피부가 예민해졌을 때 루틴을 어떻게 리셋해야 하나?

### PROOF
- CQ-PROOF-001 이 주장을 왜 믿어도 되는가?
- CQ-PROOF-002 이 설명의 근거는 무엇이며 어떤 evidence가 연결돼 있는가?
- CQ-PROOF-003 누가 이 내용을 검토했고 어떤 범위를 책임지는가?
- CQ-PROOF-004 이 정보는 언제 업데이트되었고 무엇이 바뀌었는가?

### BOUNDARY
- CQ-BOUNDARY-001 이 제품이나 성분은 누구에게는 맞지 않을 수 있나?
- CQ-BOUNDARY-002 같이 쓰면 자극이나 혼선을 만들 수 있는 조합은 무엇인가?
- CQ-BOUNDARY-004 일반 추천보다 전문가 상담이나 다른 대응이 필요한 상황은 언제인가?

### BRAND
- CQ-BRAND-001 이 브랜드는 어떤 피부 고민/상황에서 강점이 있는가?
- CQ-BRAND-002 이 브랜드의 특정 제품은 누구에게 맞고 누구에게는 맞지 않는가?
- CQ-BRAND-004 이 브랜드가 말하는 핵심 claim의 근거와 reviewer는 무엇인가?

### ACTION
- CQ-ACTION-001 지금 내 상황에서 어떤 1~3개 제품부터 시작하는 것이 맞나?
- CQ-ACTION-002 지금은 바로 구매하는 게 맞나, 상담/체험이 먼저인가?
- CQ-ACTION-004 다음 행동을 어떤 브랜드/제품/전문가 경로로 handoff해야 하나?

---

## 4. 운영 규칙

- 각 CQ는 하나의 대표 판단 장면을 가져야 한다
- primary object type이 명확해야 한다
- high-risk CQ는 boundary linkage가 필수다
- CQ는 scene으로 분해될 수 있어야 한다
